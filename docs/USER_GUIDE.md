# Guide: Plutus Cheatsheet Generator

[← Back to Documentation Portal](./index.html)

This guide covers the features and tools available in the Plutus Cheatsheet Generator.

---

## 1. Discovery

The platform provides several ways to find smart contract examples.

### 🔍 Search
The search bar at the top of the homepage filters snippets by:
- **Title**: e.g., "Vesting", "Auction".
- **Identifier**: e.g., "fortytwo", "doublesatisfaction".
- **Keywords**: Search through descriptions and content.

### 🎭 Filters
Refine results using these filters:
1. **Plutus Version**: `Plutus V1`, `Plutus V2`, or `Plutus V3`.
2. **Complexity**: `Beginner`, `Intermediate`, `Advanced`, or `Expert`.
3. **Use Case**: `DeFi`, `NFTs`, `Oracle`, `Security`, or `Utility`.

---

## 2. Tutorials

Security articles are structured as tutorials to help with debugging and identifying vulnerabilities.

### Article Structure
- **Meta Info**: Version, complexity, and use-case tags.
- **Intro**: Conceptual explanation of the contract or vulnerability.
- **On-Chain Code**: Plutus source code (Haskell).
- **Off-Chain Logic**: Bash CLI scripts or transaction examples.
- **The "Fix"**: For security guides, this explains how to mitigate the exploit.

---

## 3. Copying & Exporting

Built-in tools for managing and using snippets in your own projects.

### 📋 Copying Code
Click the **Copy** button on any code block to copy the raw source to your clipboard.

### 📥 Exporting
Download articles in two formats:
1. **Markdown (.md)**: Includes code, metadata, and source URL.
2. **PDF (.pdf)**: Print-ready version of the code block.
