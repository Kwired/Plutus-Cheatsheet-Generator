## User Guide: Browsing UI and Exporting Snippets

This section explains how to use the Plutus Cheatsheet Generator web interface.

---

### Browsing Snippets

- Open the web application in your browser
- The homepage displays a list of available Plutus snippets
- Scroll through the list to explore different examples
- Click on a snippet to view its details and source code

---

### Searching Snippets

- Use the search input at the top of the page
- Search by snippet title or identifier
- The results update automatically based on the entered keyword

---

### Exporting Snippets

Each snippet can be exported for offline reference.

- Open a snippet from the list
- Use the export options provided in the UI
- Choose **PDF** or **Markdown** format
- The file will be downloaded to your local system

No additional configuration is required to browse or export snippets.
